## 模板四：分歧形态复盘（详细）
### 1) 底层定义
分歧不是风险，是“定价权重新分配的过程”；真正危险的是分歧方式选错。

### 2) 四大类分歧
| 类型 | 本质 | 是否可博 |
|---|---|---|
| 健康分歧 | 洗不坚定筹码 | 可博 |
| 结构分歧 | 层级切换 | 克制 |
| 末期分歧 | 出货前摇 | 不博 |
| 伪分歧 | 资金撤退 | 不博 |

### 3) 可量化的分歧判断维度
1) 位置（第几板/相对高度）
2) 量能（放量 or 缩量）
3) 时间（分歧发生在何时）
4) 承接（谁在接/怎么接）
5) 环境（情绪阶段/主线状态）

### 4) 四类分歧量化标准（摘要）
- 健康分歧：2–4板/次龙；缩量或温和放量；早盘/盘中；快速回封；主线明确且情绪非退潮
- 结构分歧：龙头震荡、次龙尝试补位；同题材强弱分化
- 末期分歧：≥5板高位首次大分歧、历史最大量、高开低走/反抽无力、板上反复开板
- 伪分歧：低位无人承接、主线内集体缩量走弱、分歧后无修复尝试

### 5) 分歧量化打分模型
| 项目 | 分值 |
|---|---|
| 位置合理 | 0/1/2 |
| 量能健康 | 0/1/2 |
| 承接明确 | 0/1/2 |
| 环境支持 | 0/1/2 |
| 梯队未断 | 0/1/2 |

| 分数 | 结论 |
|---|---|
| 8–10 | 可重仓 |
| 6–7 | 轻仓博 |
| 4–5 | 观察 |
| ≤3 | 直接放弃 |

### 6) 买卖点SOP
- 买点：只买“分歧转一致”的瞬间，不提前、不预测
- 卖点：分歧未修复→次日无条件走；修复失败→盘中走；形态升级→立即走
