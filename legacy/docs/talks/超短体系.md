# 超短体系：从交割单、情绪周期到执行规则

> 说明：本文是对公开游资交割单、龙虎榜资料、游资心法，以及本目录下量价/情绪笔记的系统化整理。网上交割单存在真伪、截取样本、事后解释等问题，所以这里提炼的是“可复用的交易结构”，不是复制某个账户的买卖点。本文不构成投资建议。

## 0. 先定义：超短到底赚什么钱

超短不赚“公司长期价值”的钱，也不应该幻想赚“提前知道消息”的钱。

超短主要赚三类钱：

1. 情绪扩散的钱：市场从不信到信，资金开始集中。
2. 承接确认的钱：分歧出现后，有更强资金把抛压接住。
3. 流动性溢价的钱：核心票被市场反复关注，次日、隔日仍有人愿意接。

所以超短的第一性原理不是题材，而是：

```text
资金集中度 + 赚钱效应 + 承接强度
```

题材只是资金行动的理由。主线也是结果，不是起点。

## 1. 资料分层：哪些能学，哪些只能参考

### 1.1 公开交割单的共性

能学的不是“某天买了什么”，而是这些东西：

- 买入前，市场已经出现可观察的强度。
- 买入时，往往是放量、涨停、回封、弱转强、突破等确认点。
- 持仓周期短，错误处理快。
- 大肉通常来自少数核心交易，其他多是小赚、小亏、试错。
- 真正的大成不靠单月暴利，而靠长期控制回撤后的复利。

### 1.2 不可直接照抄的地方

- 交割单可能不完整，只展示好看的阶段。
- 交易者的资金体量、通道、席位影响力、信息处理能力不同。
- 过去市场生态和现在不同，量化、监管、20cm、注册制、社媒反噬都会改变接力环境。
- 只看买卖点，不看当时市场周期，基本等于看无声电影。

### 1.3 几类样本的差异

| 样本 | 可提炼重点 | 不能直接照抄 |
| --- | --- | --- |
| 炒股养家 | 情绪周期、弱市不做、主流/赚钱效应、大局观 | 早期交割单所处生态、资金规模较小 |
| 赵老哥类实盘 | 高换手、高频试错、快速纠错、长期复利 | 不能简化成“只做龙头” |
| 余哥类新生代 | 主流大成交、人气核心、趋势持股、重仓进攻 | 大仓位波动巨大，公开资料受社媒叙事影响 |
| 赵晨/东北游资类 | 大资金高举高打、板块/个股兵团作战、敢买敢卖 | 公开完整交割单少，更多是席位风格描述 |
| 本地 docs/talks | 量价循环、分歧/出货、情绪拐点、资金结构 | 偏模型，需要落到纪律和执行 |

## 2. 共性经验：所有材料最后都指向同一套骨架

### 2.1 做核心，不做杂毛

核心不是你喜欢的票，而是市场愿意反复给流动性和溢价的票。

判断核心看五点：

```text
1. 是否最早启动
2. 是否带动板块
3. 是否分歧后还能修复
4. 是否成交额足够容纳资金
5. 是否次日仍有溢价
```

做不到这五点，哪怕题材再美，也只是跟风。

### 2.2 强弱由盘面确认，不由观点确认

超短最忌“我觉得它应该涨”。

你只问：

```text
资金是否正在买
买完是否封住
封住后是否有溢价
分歧时是否有人接
```

如果没有，就不是机会，只是想象。

### 2.3 分歧是机会，出货是陷阱

同样是放量，结果完全不同。

| 类型 | 盘面 | 结论 |
| --- | --- | --- |
| 健康分歧 | 放量、震荡、回封、收盘强 | 可以参与 |
| 出货 | 放量、长上影、反复炸板、收盘弱 | 回避 |
| 顶部衰竭 | 巨量但涨不动，次日无溢价 | 降仓或空仓 |
| 底部修复 | 缩量不跌，出现抗跌核心，放量首板 | 小仓试错 |

关键句：

```text
分歧必须被吃掉，否则就是风险释放没有完成。
```

### 2.4 弱市不做，是最高级的进攻

弱市不是没有涨停，而是赚钱效应不持续。

弱市特征：

- 昨日涨停次日无溢价。
- 高位票开始 A 杀。
- 连板高度下降。
- 每天一个热点，次日直接熄火。
- 放量冲高但封不住。

弱市的正确动作不是寻找神票，而是：

```text
空仓为主，最多小仓观察抗跌核心。
```

### 2.5 大赚靠少数机会，小亏靠纪律

成熟超短账户的资金曲线不是天天暴涨，而是：

```text
多数时间小赚小亏横住
少数核心机会快速拉升
错误时不让亏损扩大
```

所以系统的重点不是提高每笔胜率到极致，而是：

```text
亏损可控 + 盈利时敢拿 + 节点来时敢上
```

## 3. 市场周期：每天先判断能不能做

把市场压缩成五个阶段：

```text
冰点
→ 试错
→ 主升
→ 高潮
→ 退潮
→ 再冰点
```

### 3.1 冰点

特征：

- 跌停、亏钱效应释放。
- 成交量下降。
- 高位票大面积补跌。
- 市场极度悲观。

动作：

```text
不急着抄底，观察谁抗跌。
```

### 3.2 试错

特征：

- 出现放量首板。
- 次日开始有溢价。
- 抗跌票走强。
- 题材轮动仍快。

动作：

```text
小仓试错，只做第一个走出来的。
```

### 3.3 主升

特征：

- 主线清晰。
- 龙头持续新高。
- 分歧后能修复。
- 后排有补涨，板块有梯队。

动作：

```text
集中做核心，分歧确认后上仓位。
```

### 3.4 高潮

特征：

- 一致加速。
- 缩量一字、秒板增多。
- 后排乱涨。
- 所有人都知道主线。

动作：

```text
不追后排，不在高潮次日盲目接力，核心只做分歧后的确认。
```

### 3.5 退潮

特征：

- 龙头断板后不能修复。
- 高位票放量大跌。
- 昨日涨停次日亏钱。
- 新题材一日游。

动作：

```text
降仓或空仓，等待下一轮抗跌核心。
```

## 4. 标的选择：只选四类机会

### 4.1 低位启动

适用阶段：

```text
冰点后期、试错期、主升初期
```

量价路径：

```text
缩量横盘
→ 放量首板
→ 缩量/平量二板
→ 三板放量回封
```

参与条件：

- 首板不能是一字无量。
- 二板要强，最好缩量或平量快速封板。
- 三板分歧必须能回封。
- 次日要有溢价。

放弃条件：

- 首板无板块带动。
- 二板爆量烂板。
- 三板放量炸板回落。
- 次日低开低走。

### 4.2 龙头分歧

适用阶段：

```text
主升期、强修复期
```

参与条件：

- 龙头地位明确。
- 板块仍有梯队。
- 分歧当天放量但不崩。
- 回封质量强，或日内承接明显强于同类。

放弃条件：

- 龙头第一次断板后板块全散。
- 放量长上影，收盘弱。
- 次日无修复。
- 市场整体进入退潮。

### 4.3 弱市抱团

适用阶段：

```text
弱市、无主线、指数震荡
```

参与条件：

- 指数跌，它横住或上涨。
- 成交额持续放大。
- 每次分歧都被接住。
- 市场开始把它当作锚点。

动作：

```text
只做最抗跌的 1-2 只，不能扩散到一堆跟风。
```

### 4.4 二波/反包

适用阶段：

```text
老主线未死，情绪修复
```

参与条件：

- 前一波下跌是缩量，不是放量出货。
- 调整阶段不创新低，有承接。
- 反包当天放量突破关键压力或前高。
- 板块或市场情绪同步修复。

放弃条件：

- 高位放量下跌后直接反包。
- 缩量涨停但未突破压力。
- 只是老资金自救，没有新增合力。
- 反包次日无溢价。

## 5. 买点：只做确认，不做幻想

### 5.1 打板买点

适合：

- 低位启动确认。
- 龙头分歧回封。
- 板块共振首日。

要求：

```text
放量充分 + 封单稳定 + 板块配合 + 次日有预期
```

不打：

- 缩量一字开板后的情绪接盘。
- 尾盘偷板。
- 后排跟风板。
- 市场退潮期的孤立板。

### 5.2 低吸买点

适合：

- 龙头分歧日。
- 弱市抗跌核心。
- 主线趋势票回踩。

要求：

```text
位置是核心，回踩不破关键位，盘口承接强于板块
```

不低吸：

- 下跌趋势里的普通反弹。
- 高位放量出货后的“便宜”。
- 没有市场地位的后排。

### 5.3 弱转强买点

适合：

- 前一日分歧充分。
- 次日竞价或开盘超预期。

要求：

```text
昨天该弱不弱，今天该低不开低，开盘快速转强
```

注意：

弱转强必须发生在核心票上。杂毛弱转强只是诱多概率高。

## 6. 仓位：仓位只奖励确认，不奖励冲动

一个简单可执行的仓位模型：

| 情况 | 仓位 |
| --- | --- |
| 冰点/退潮 | 0 |
| 试错期首个核心 | 10%-20% |
| 主升期核心分歧回封 | 30%-50% |
| 已有利润垫且次日继续超预期 | 可加到 50%-70% |
| 高潮后、退潮初 | 降到 0%-20% |

加仓规则：

```text
只有持仓已经盈利，且市场继续确认，才允许加仓。
```

减仓规则：

```text
只要走势偏离买入预期，先降风险，再谈判断。
```

## 7. 卖点：错误不过夜，高潮不贪恋

### 7.1 必卖

- 买入逻辑当天被证伪。
- 打板炸开后无法回封。
- 龙头变后排。
- 板块核心先于自己走弱。
- 次日无溢价且开盘不能修复。
- 高位巨量长上影，收盘弱。

### 7.2 可拿

- 龙头分歧后回封。
- 板块梯队完整。
- 次日高开后承接强。
- 高位换手充分但收盘强。
- 自己的买点低，已有利润垫。

### 7.3 分批

适合：

- 龙头主升。
- 二波突破。
- 趋势核心。

原则：

```text
冲高先落袋一部分，强势再留核心仓观察。
```

## 8. 风控：系统能活下来，机会才有意义

### 8.1 单笔风控

建议规则：

```text
单笔最大亏损不超过账户 1%-2%
```

换算方式：

```text
可承受账户亏损 / 个股止损幅度 = 最大仓位
```

例如：

```text
账户最多亏 1%
个股止损 5%
最大仓位 = 20%
```

### 8.2 日内风控

- 当日连续两笔失败，停止开新仓。
- 当日亏损超过预设线，停止交易。
- 市场退潮确认后，不做“回本单”。

### 8.3 周期风控

- 连续三天无赚钱效应，仓位降到观察仓。
- 连续出现高位 A 杀，暂停接力。
- 连续两次模式内亏损，降低仓位复盘，不改系统追别的模式。

## 9. 每天执行流程

### 9.1 盘前

只回答五个问题：

```text
1. 市场处于冰点、试错、主升、高潮、退潮哪一段？
2. 昨日涨停今天是否有溢价？
3. 当前最高板/核心趋势票是谁？
4. 主线是否有梯队？
5. 今天只允许做哪 1-3 个标的？
```

如果答不出来：

```text
今天默认不做。
```

### 9.2 开盘 5-15 分钟

观察：

- 核心票是否超预期。
- 昨日强势股是否给溢价。
- 昨日弱票是否继续核。
- 新题材是板块共振，还是单票脉冲。
- 成交额是否支持今天继续进攻。

动作：

```text
只做计划内标的，只做计划内买点。
```

### 9.3 盘中

只盯三件事：

```text
封板质量
板块梯队
核心承接
```

少看消息，多看资金反馈。

### 9.4 收盘后

复盘模板：

```text
日期：
市场阶段：
主线：
核心票：
赚钱效应：
亏钱效应：
今日交易：
买入理由：
卖出理由：
是否模式内：
如果重来一次：
明日计划：
```

## 10. 一页决策树

```text
第一步：市场能不能做？

退潮/亏钱效应扩散
→ 空仓

冰点后期/试错期
→ 小仓做第一个放量核心

主升期
→ 做主线核心，分歧确认后上仓位

高潮期
→ 不追后排，核心冲高减仓，等分歧

第二步：标的是不是核心？

不是核心
→ 不做

是核心
→ 看买点

第三步：买点是否确认？

放量首板/二板确认
→ 可小仓

龙头分歧回封
→ 可中仓

弱转强超预期
→ 可跟随

高位放量收弱
→ 不做

第四步：买后是否符合预期？

符合
→ 拿或加

不符合
→ 减或走
```

## 11. 最简口诀

```text
有资金集中，做核心。
无资金集中，等。
轮动期，盯抗跌。
新题材，只做第一个走出来的。
分歧被吃掉，才是机会。
放量涨不动，就是风险。
弱市不做，错了就走。
```

## 12. 和本地 docs/talks 的对应关系

- `逻辑.md`：把题材框架升级为资金集中度、赚钱效应、承接强度。
- `轮回.md`：提供完整量价循环，说明行情如何从启动到顶部再到衰竭。
- `情绪.md`：提供情绪拐点识别，尤其是量价失配。
- `轮回短.md`：区分低位连板和高位反包，避免把“续命”当“新启动”。

本体系是在这四份笔记基础上补了三块：

```text
1. 资料可信度过滤
2. 仓位与风控
3. 每日执行流程
```
