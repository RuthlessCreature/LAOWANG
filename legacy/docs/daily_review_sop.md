

# 超短复盘 SOP · 模块一

## 市场情绪量化判断体系（完整版）

---

## 一、情绪的本质定义（先统一口径）

> **市场情绪 = 风险偏好 × 赚钱效应 × 容错率**

不是涨跌，不是指数，是**资金是否愿意为“不确定性”付溢价**。

---

## 二、情绪量化指标总表（你每天必须算）

### ① 核心七指标（一级指标）

| 指标      | 含义     | 数据口径       |
| ------- | ------ | ---------- |
| 涨停家数    | 正向赚钱样本 | 当日非 ST 涨停  |
| 跌停家数    | 风险释放   | 非 ST 跌停    |
| 炸板率     | 容错率    | 炸板数 / 封板数  |
| 连板高度    | 情绪上限   | 最高非 ST 连板  |
| 昨日涨停红盘率 | 情绪延续   | 昨涨停次日收红    |
| 昨日炸板红盘率 | 情绪修复   | 昨炸板次日收红    |
| 成交额变化   | 增量资金   | 今日 vs 5 日均 |

---

## 三、每个指标的“情绪分档标准”（关键）

### 1️⃣ 涨停家数（赚钱效应）

| 数值    | 解释   |
| ----- | ---- |
| ≥80   | 情绪高涨 |
| 50–79 | 情绪可用 |
| 30–49 | 情绪偏弱 |
| <30   | 情绪冰点 |

📌 **低于 30，超短不主动出击。**

---

### 2️⃣ 跌停家数（风险释放）

| 数值    | 解释  |
| ----- | --- |
| 0–3   | 风险低 |
| 4–10  | 可控  |
| 11–20 | 警惕  |
| >20   | 恐慌  |

📌 **跌停不是坏事，未释放才危险。**

---

### 3️⃣ 炸板率（容错率核心）

计算：
**炸板率 = 炸板家数 ÷（炸板 + 封死涨停）**

| 比例     | 解释   |
| ------ | ---- |
| <25%   | 容错极高 |
| 25–40% | 正常   |
| 40–55% | 容错下降 |
| >55%   | 情绪退潮 |

📌 **超短最怕的不是跌，是“不让涨”。**

---

### 4️⃣ 连板高度（空间上限）

| 高度  | 情绪状态    |
| --- | ------- |
| ≥7  | 主升 / 高潮 |
| 5–6 | 强势      |
| 3–4 | 修复      |
| ≤2  | 冰点      |

📌 **高度压制 = 所有接力失败的根源。**

---

### 5️⃣ 昨日涨停次日红盘率（延续性）

| 比例     | 解释   |
| ------ | ---- |
| ≥65%   | 情绪健康 |
| 50–64% | 勉强   |
| 35–49% | 弱    |
| <35%   | 断层   |

📌 **这是最容易被忽视、却最诚实的指标。**  

---

### 6️⃣ 昨日炸板次日红盘率（修复性）

| 比例     | 解释   |
| ------ | ---- |
| ≥65%   | 修复健康 |
| 50–64% | 勉强   |
| 35–49% | 偏弱   |
| <35%   | 断层   |

📌 **炸板是“情绪容错”的试金石，次日是否转红是修复力核心。**

---

### 7️⃣ 成交额变化（增量判断）

| 状态      | 解释   |
| ------- | ---- |
| 放量 ≥10% | 有增量  |
| 持平      | 存量博弈 |
| 缩量 ≥10% | 风险上升 |

📌 **没有增量，高潮必死。**

---

| 跌停家数 | 低 / 中 / 高 | +2 / +1 / 0 |

| 连板高度 | 高 / 中 / 低 | +2 / +1 / 0 |
| 红盘率  | 高 / 中 / 低 | +2 / +1 / 0 |
| 昨日炸板红盘率 | 高 / 中 / 低 | +2 / +1 / 0 |

### 总分 → 情绪阶段

| 0–5   | 冰点   |

📌 **这个判断结果，是你明天仓位的“硬约束”。**

# 超短复盘 SOP · 模板二

## 主线题材的量化识别与筛选体系（完整版）

---

## 一、先给结论（防止跑偏）

> **超短里的“主线”，不是行业趋势，不是逻辑最硬，
> 而是：当下最集中、最有持续性的情绪承载体。**

主线 = **资金共识的最大公约数**

---

## 二、主线的严格定义（量化口径）

一条题材要被认定为【主线】，**必须同时满足以下 4 条中的 ≥3 条**：

| 维度 | 定义          |
| -- | ----------- |
| 广度 | 涨停/强势股数量    |
| 高度 | 连板空间是否打开    |
| 持续 | 是否多日反复被资金选择 |
| 排他 | 是否压制其他题材    |

📌 不满足 = **轮动 / 噪音**

---

## 三、主线量化指标拆解（核心）

### ① 题材广度（Breadth）

统计口径（当日）：

* 题材内 **涨停家数**
* 题材内 **涨幅 ≥5% 家数**
* 是否出现 **梯队结构**

> 口径补充（实现建议）：
> - 题材归属优先使用涨停池自带的 `related_plates/概念/题材` 字段聚合
> - 统计口径建议剔除 ST

#### 分档标准

| 广度表现        | 结论   |
| ----------- | ---- |
| ≥5 涨停 + 多梯队 | 主线候选 |
| 3–4 涨停      | 次主线  |
| ≤2 涨停       | 轮动   |
| 单票独舞        | 假题材  |

📌 **没有广度的“龙头”，只是庄股。**

---

### ② 题材高度（Height）

你只关心一件事：
**这条题材有没有“往上打”的空间许可？**

| 最高板 | 解读   |
| --- | ---- |
| ≥5  | 空间打开 |
| 3–4 | 修复阶段 |
| ≤2  | 无高度  |

📌 **高度不打开，所有接力都是抢跑。**

---

### ③ 题材持续性（Persistence）

统计最近 **3–5 个交易日**：

* 是否 **反复成为涨停最多的题材**
* 是否 **退潮后有资金回流**
* 是否 **核心票始终有人维护**

#### 判断标准

| 表现         | 定性  |
| ---------- | --- |
| 连续 2 天以上强势 | 强主线 |
| 隔日轮动       | 弱主线 |
| 一日游        | 非主线 |

📌 **超短不做“第一次”，只做“被验证的第二次”。**

---

### ④ 题材排他性（Exclusivity）

这是**最容易被忽视，但权重极高**的一项。

你要观察：

* 是否出现 **其他题材被集体压制**
* 非主线个股 **高开低走**
* 市场注意力是否高度集中

| 排他表现     | 结论  |
| -------- | --- |
| 明显压制其他题材 | 真主线 |
| 多线并行     | 混沌  |
| 你强我也强    | 无主线 |

📌 **钱不够的时候，主线一定是“独裁制”。**

---

## 四、主线判定打分模型（强制）

每条题材每天打分：

| 维度  | 分值        |
| --- | --------- |
| 广度  | 0 / 1 / 2 |
| 高度  | 0 / 1 / 2 |
| 持续性 | 0 / 1 / 2 |
| 排他性 | 0 / 1 / 2 |

### 总分 → 定位

| 总分  | 定义       |
| --- | -------- |
| 7–8 | **唯一主线** |
| 5–6 | 主线       |
| 3–4 | 次主线      |
| ≤2  | 轮动 / 垃圾  |

📌 **每天你只能承认一个 7–8 分的题材。**

---

## 五、主线 ≠ 可交易（关键过滤）

即便是主线，也要再过 **3 个交易性过滤器**：

### 1️⃣ 是否“末期主线”

危险信号：

* 高位一致
* 连板一字
* 题材被过度解读

📌 **末期主线只看不做。**

---

### 2️⃣ 是否“高度压制”

* 龙头打到 6–7 板
* 次龙永远 2–3 板失败

📌 **有主线，但没接力空间。**

---

### 3️⃣ 是否“监管敏感”

* 涉嫌炒作红线
* 盘后点名、问询

📌 **监管 > 情绪**

# 超短复盘 SOP · 模板三

## 龙头 / 梯队结构的量化识别与博弈体系（完整版）

---

## 一、先下结论（避免认知错误）

> **超短不是选“好票”，是选“话事人”。**
> 龙头不是涨得最多的，是**能决定其他票生死的那一只**。

---

## 二、梯队的标准定义（统一口径）

在一条有效主线中，股票只分四类：

| 层级 | 定义      |
| -- | ------- |
| 龙头 | 情绪锚，定价权 |
| 次龙 | 接力主力    |
| 补涨 | 博弈品     |
| 炮灰 | 出货工具    |

📌 **你只能在前两层投入重仓。**

---

## 三、龙头的量化识别标准（硬条件）

一只票要被认定为【龙头】，**必须同时满足 ≥4 条**：

### ① 空间优势（最基础）

* 主线内 **最高连板**
* 或 **唯一打破前高**

> 没空间 = 没资格当龙头

---

### ② 时间优势（先发性）

* 最早启动
* 或第一批封板

📌 **后发只能当次龙。**

---

### ③ 资金一致性（关键）

量化观察点：

* 封单 **反复被砸仍能回封**
* 分歧日 **不爆量**
* 多日 **换手递增而非衰减**

📌 **龙头允许分歧，不允许失控。**

---

### ④ 情绪代表性（最重要）

你只问一个问题：

> **它动的时候，其他同题材票是不是跟着动？**

* 是 → 龙头
* 否 → 强票而已

---

### ⑤ 指数脱钩能力（加分项）

* 指数弱，它不弱
* 板块分化，它不死

📌 **真龙必然抗指数。**

---

## 四、次龙的量化识别（你主要的战场）

> **80% 的稳定利润，来自次龙。**

### 次龙必须满足：

| 维度 | 要求         |
| -- | ---------- |
| 地位 | 主线第二或第三    |
| 换手 | 充分，不一字     |
| 空间 | 比龙头低 1–2 板 |
| 跟随 | 龙头强时必响应    |

📌 **次龙是“性价比最高”的交易对象。**

---

## 五、梯队健康度评估（关键）

你每天必须评估主线梯队是否“还能玩”。

### 健康梯队特征

* 龙头稳住
* 次龙能接
* 补涨有人尝试
* 炮灰快速淘汰

### 崩塌梯队信号（任一即危险）

* 龙头断板
* 次龙集体失败
* 补涨全死
* 炮灰数量激增

📌 **梯队一旦断层，主线寿命进入倒计时。**

---

## 六、梯队量化打分模型（强制）

对主线梯队整体打分：

| 项目     | 分值        |
| ------ | --------- |
| 龙头稳定性  | 0 / 1 / 2 |
| 次龙承接力  | 0 / 1 / 2 |
| 补涨尝试   | 0 / 1 / 2 |
| 失败淘汰效率 | 0 / 1 / 2 |

### 总分 → 策略

| 分数  | 结论      |
| --- | ------- |
| 7–8 | 主升期，可重仓 |
| 5–6 | 可做，控仓   |
| 3–4 | 边走边看    |
| ≤2  | 准备撤退    |

---

## 七、不同梯队的“允许打法”（铁律）

### 龙头

* 只做：**换手接力 / 分歧转一致**
* 不做：追一致高潮

### 次龙（重点）

* 分歧低吸
* 回踩承接
* 次日溢价走

### 补涨

* 只做首板 / 1→2
* 失败不留恋

### 炮灰

* 不碰

# 超短复盘 SOP · 模板四

## 分歧形态的量化分类、统计与博弈模型（完整版）

---

## 一、先给底层定义（防止走偏）

> **分歧不是风险，是“定价权重新分配的过程”。**
> 真正危险的不是分歧，而是**分歧方式选错**。

超短里 **90% 的亏损**，都来自一句话：

> *“我以为这是正常分歧。”*

---

## 二、分歧的四大类（先定性）

在主线 + 梯队已确认的前提下，分歧只分四种：

| 类型   | 本质     | 是否可博 |
| ---- | ------ | ---- |
| 健康分歧 | 洗不坚定筹码 | ✅    |
| 结构分歧 | 层级切换   | ⚠️   |
| 末期分歧 | 出货前摇   | ❌    |
| 伪分歧  | 资金撤退   | ❌    |

你每天复盘的目标，是**把当日所有分歧归类**。

---

## 三、可量化的分歧判断维度（核心）

任何一只票出现分歧，你只允许从 **5 个维度**判断：

1. **位置**（第几板 / 相对高度）
2. **量能**（放量 or 缩量）
3. **时间**（分歧发生在什么时候）
4. **承接**（谁在接、怎么接）
5. **环境**（情绪阶段 / 主线状态）

📌 **缺任何一项，判断无效。**

> 口径补充（实现建议）：
> - 量能/承接优先使用分钟线（5/15/30/60 分钟）判断分歧发生区间与回封速度

---

## 四、四类分歧的量化标准（重点）

---

## ① 健康分歧（唯一值得重仓的）

### 量化特征（必须同时满足 ≥4 条）

| 维度 | 标准          |
| -- | ----------- |
| 位置 | 2–4 板 / 次龙  |
| 量能 | 缩量或温和放量     |
| 时间 | 早盘或盘中       |
| 承接 | 快速回封 / 明确大单 |
| 环境 | 主线明确，情绪非退潮  |

### 典型盘口形态

* 高开 → 下杀 → 稳住 → 回封
* 或低开 → 快速拉起 → 换手封板

📌 **这是“洗筹码”，不是“跑资金”。**

---

## ② 结构分歧（需要极强克制）

> 本质：**钱没走，但在换人**

### 量化特征

* 龙头开始震荡
* 次龙、低位票尝试补位
* 同题材 **强弱分化**

| 判断点    | 含义     |
| ------ | ------ |
| 龙头未死   | 主线仍在   |
| 次龙成功率  | 决定是否参与 |
| 梯队是否断层 | 决定能否继续 |

📌 **只做“确认后的胜者”，不赌切换瞬间。**

---

## ③ 末期分歧（杀伤力最大）

### 危险信号（满足任一，直接归类）

* ≥5 板高位首次大分歧
* 放出历史最大量
* 高开低走、反抽无力
* 板上反复开板，封不住

📌 **末期分歧 = 出货信号，不是机会。**

## ④ 伪分歧（很多人会误判）

> 看起来像分歧，实际是**资金不玩了**

### 特征

* 低位票分歧但无人承接
* 主线内 **集体缩量走弱**
* 分歧后无任何修复尝试

📌 **这是退潮的前奏。**

## 六、分歧 → 买卖点 SOP（极重要）

### 买点铁律

* **只买“分歧转一致”的瞬间**
* 不提前，不预测

### 卖点铁律

* 分歧未修复 → 次日无条件走
* 修复失败 → 盘中走
* 分歧形态升级 → 立即走

📌 **超短卖点永远比买点重要。**
