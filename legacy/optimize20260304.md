## 四、rules.txt 优化建议

### 当前规则
```
训练集: 2020-01-01 至 2024-12-31
回测集: 2025-01-01 至 2026-02-28
最大回撤: ≥15%
收益率: ≥1000%
```

### 建议调整

1. **提高最低收益要求**: 从 1000% 提升到 1500%
2. **降低最大回撤**: 从 15% 降低到 12%
3. **增加手续费**: 考虑实际滑点，手续费调整为 0.03%
4. **添加止损规则**: 单日最大亏损 5% 止损
