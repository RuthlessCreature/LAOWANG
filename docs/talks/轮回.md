把整段行情压成一个**从低位→高位→再回落的量价循环模型**，你只需要记住：

**每一段的“量的变化逻辑”完全不同。**

---

# 总框架（先建立骨架）

```
低位吸筹
→ 启动放量
→ 连板/趋势（缩量）
→ 分歧放量（关键）
→ 加速缩量
→ 顶部放量
→ 下跌（放量→缩量）
→ 再平衡
```

这是一个闭环。

---

# 一、低位：吸筹区（无趋势）

## 量价特征

```
价格：横盘
成交量：不稳定，小幅放大
```

关键细节：

```
阳线放量
阴线缩量
```

含义：

```
有人在慢慢收筹码
```

👉 这个阶段：

- 没趋势
- 没情绪
- 但有“资金痕迹”

---

# 二、启动点：第一次放量（最重要的起点）

## 量价特征

```
放量大阳 / 涨停
换手 10%-20%
```

含义：

```
资金开始主动进攻
```

注意：

```
这是“从无到有”的跃迁
```

---

# 三、初期趋势：缩量上涨（健康阶段）

## 量价特征

```
价格持续上涨
成交量：缩量或平量
```

含义：

```
抛压很轻
筹码锁定
```

👉 这是最舒服的阶段：

```
不需要很多钱，就能涨
```

---

# 四、第一次分歧：放量（关键节点）

这是整个循环最关键的位置。

## 量价特征

```
成交量突然放大
价格剧烈波动（炸板/长上影）
```

含义：

```
有人开始卖
有人继续接
```

---

## 分叉点来了：

### ▶ A：放量后还能涨

```
分歧被吃掉
```

→ 进入更强趋势

---

### ▶ B：放量后涨不动

```
承接不足
```

→ 见顶前兆

---

# 五、加速阶段：再次缩量（危险）

很多人误判在这里。

## 量价特征

```
连续上涨
成交量反而减少
甚至一字板
```

含义：

```
情绪高度一致
```

问题在于：

```
没有换手
```

👉 结论：

```
越缩量 → 越脆弱
```

---

# 六、顶部：巨量（情绪高潮）

## 量价特征

```
成交量极大（阶段最大）
换手 30%-40%
K线剧烈震荡
```

含义：

```
所有人都来了
```

资金结构：

| 人群 | 行为 |
| --- | --- |
| 早期资金 | 卖 |
| 中期资金 | 犹豫 |
| 后知后觉 | 买 |

👉 本质：

```
筹码大转移
```

---

# 七、下跌初期：放量下跌（危险区）

## 量价特征

```
下跌 + 放量
```

含义：

```
资金在撤退
```

👉 这是最该回避的阶段。

---

# 八、下跌后期：缩量下跌（接近底部）

## 量价特征

```
下跌但量越来越小
```

含义：

```
想卖的人已经卖完
```

👉 这里才开始有机会。

---

# 九、重新平衡（准备下一轮）

表现：

```
横盘
偶尔放量
```

→ 回到第一阶段

---

# 把整个过程压缩成“量价节奏表”

| 阶段 | 价格 | 成交量 | 含义 |
| --- | --- | --- | --- |
| 低位 | 横盘 | 小波动 | 吸筹 |
| 启动 | 突破 | 放量 | 点火 |
| 初期趋势 | 上涨 | 缩量 | 锁筹 |
| 分歧 | 波动 | 放量 | 换手 |
| 加速 | 上涨 | 再缩量 | 一致 |
| 顶部 | 震荡 | 巨量 | 出货 |
| 下跌初期 | 下跌 | 放量 | 撤退 |
| 下跌后期 | 下跌 | 缩量 | 衰竭 |

---

# 最关键的三次“放量”

你不用记全部，只记这三个点：

---

## ① 启动放量

```
决定有没有行情
```

---

## ② 分歧放量

```
决定能走多远
```

---

## ③ 顶部放量

```
决定什么时候结束
```

---

# 最容易犯错的地方

---

## 错误1

```
把“加速缩量”当强势
```

实际上：

```
那是没人接盘
```

---

## 错误2

```
把“顶部放量”当机会
```

实际上：

```
那是别人出货
```

---

## 错误3

```
在“第一次分歧”恐慌卖出
```

但真正情况是：

```
那可能是最佳买点
```

---

# 最后给你一句压缩版

```
行情靠放量启动
靠缩量上涨
死于再次放量
```

---

# 再补一刀（更底层）

```
量 = 分歧
缩量 = 一致
```

市场运行逻辑就是：

```
分歧 → 一致 → 再分歧 → 崩
```

---

如果你要再往上走一步，可以继续拆：

👉 **“同样是放量，为什么有的是机会，有的是陷阱？”**

这个点一旦打通，你基本就能从“看懂”进入“能做”。
